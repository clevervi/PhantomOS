# Security Policy

## Supported Versions
Only the latest version of PhantomOS is officially supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | ✅ (Current MVP)    |

## Reporting a Vulnerability
We take the security of your system very seriously. If you find a vulnerability, please do NOT create a public issue. 

Instead, please send an email to **security@clevervi.com** or contact the maintainers directly via GitHub. We aim to respond within 48 hours and provide a patch as soon as possible.

## Security Architecture
PhantomOS interacts with critical system components. To minimize risk:
- It requires Administrative privileges.
- All registry changes are backed up.
- Code is open for audit at any time.
